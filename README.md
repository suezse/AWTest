# Invitation

### Installing
```bash
//1. 下载代码
$ git clone https://github.com/suezse/AWTest.git

//2. 安装依赖
$ yarn install 或者 npm install

//3. 启动项目
$ npm start

//4. 启动成功则显示“Project is running at http://localhost:5001/”，
//在浏览器地址栏输入http://localhost:5001/ 打开页面（5001若被占用可以修改
//webpack.config.js文件中的port）。

```
